## Overview

All endpoints are hosted under <Code>https://ledgerize.ai/api/v1</Code> and require an API key in the
<Code>X-Api-Key</Code> header. Requests should be made over HTTPS.

```http
X-Api-Key: YOUR_LEDGERIZE_API_KEY
Content-Type: application/json
```

## Workflow

1. **Upload** bank-statement PDFs via `POST /files` (multipart form). The response includes a `uuid`
   you use for conversions.
2. **Convert** one or more UUIDs with `POST /convert`. Choose `JSON`, `CSV`, or `XLS` formats and
   Ledgerize will start an asynchronous job.
3. **Check job status** using `GET /jobs/{jobId}`. When the job is `completed`, fetch the results at
   `GET /jobs/{jobId}/result`.
